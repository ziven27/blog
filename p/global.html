【网站七步曲-第三步】css-global

前两步，我们都是可以复用的东西。在不同的项目中，只需要稍加修改就可以使用了。

后面这几步我们都需要基于项目的实际情况来构建。一般我们会从UI那里拿到我们项目的高保真的PSD文件（也不排除一些坑爹的项目给的只是一个奇丑无比的原型图）。

这一步我们就来将脱掉的样式，一件件穿回来。

拿到psd之后，我们稍微看一下，就可以知道我们当前项目的基本样式了。如下：

	html { background-color:#fff; color: #666; }
	html, input, button, textarea, select { font: caption; font-size: 14px; line-height: 20px; }
